Security Policy

Reporting Security Issues

If you discover a security vulnerability in Vocaband, please disclose it responsibly:

  • Email: security@vocaband.co.il
  • Please include details about the vulnerability and steps to reproduce
  • We will respond within 48 hours and keep you updated on our progress

Our Security Commitment

Vocaband is committed to maintaining the security and privacy of our users:

  • All data is encrypted in transit (HTTPS/TLS)
  • User authentication is handled via Supabase Auth with JWT tokens
  • Row Level Security (RLS) policies protect database access
  • Rate limiting prevents abuse and attacks
  • We use security best practices including helmet.js headers
