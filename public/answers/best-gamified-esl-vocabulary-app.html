How each app stacks up

1. Vocaband — best free pick for ESL classrooms

Built specifically for ESL teachers running real classes. Class code joining (no student emails), 10 game modes covering listening, spelling, matching, sentence-building, etc. Per-word mastery heatmap so teachers see exactly which words each student struggles with. Free.

2. Kahoot! — best for live quiz events

Unbeatable live-quiz experience with the giant projected screen + students racing to answer. But it's not built for spaced practice — you can't hand it to a student to drill 50 words solo. Best as a once-a-week class event, not a daily tool.

3. Wordwall — best for craft-it-yourself

Wide range of activity templates (crossword, hangman, balloon pop, etc.). Teachers love it for creating custom games. Cap of 5 saved games on the free plan is the catch.

4. Duolingo — best for solo learners

Excellent gamified solo-learning loop. Duolingo for Schools is improving but still very limited compared to a tool built around a teacher running a class.

5. Quizlet — best for multi-subject teachers

The classic. Strong if you teach multiple subjects (history, science, English). Free tier has narrowed significantly since 2023; teachers wanting Learn mode + Quizlet Live's full features now pay ~$36/year.

6. Memrise — best for individual progress tracking

Solid spaced-repetition engine. No meaningful classroom mode.

Decision framework — which to pick

  • You teach ESL and want one tool your whole class uses → Vocaband.
  • You want live in-class events twice a week → Kahoot!.
  • You want to build custom activities yourself → Wordwall.
  • You teach mixed subjects → Quizlet.
  • You're a solo learner, not a teacher → Duolingo or Memrise.
